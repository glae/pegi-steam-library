# PEGI on steam games library

## purpose

Parse a game list (e.g. from steam) as input and prints its [PEGI info](https://pegi.info) in a HTML output. 

 
## On Ubuntu/Linux

### launch app

    sbt "run my-games.txt"


### unit test 

    sbt test
    

### install dev environment 

    ./__setup.sh
    
### uninstall dev environment 

    ./__uninstall.sh
